# Python Data Science Assignment 02

This repository contains solutions for **Assignment 02** from the **PythonX Bootcamp - Python Foundations for Data Science**.

The assignment focuses on practical applications of **NumPy**, **TensorFlow**, and **Matplotlib** in healthcare, research, and AI-related scenarios.

## Assignment Topics

### Question 1: Patient Vitals Analysis with NumPy
- Creating a 1D NumPy array of patient heart rate readings
- Reshaping arrays
- Computing mean, max, and min values
- Element-wise operations and broadcasting

### Question 2: Student Exam Score Analysis
- Creating a 2D NumPy matrix
- Array slicing
- Column indexing
- Boolean indexing for filtering scores

### Question 3: TensorFlow Tensor Operations
- Creating tensors using `tf.constant()`
- Checking tensor shape and data type
- Reshaping tensors
- Using `tf.expand_dims()` and `tf.transpose()`
- Comparing TensorFlow tensors with NumPy arrays

### Question 4: Medical Image Processing & Visualization
- Loading and displaying images
- Converting images to grayscale
- Plotting pixel intensity histogram
- Comparing original and grayscale images
- Saving visualizations as `.png`
